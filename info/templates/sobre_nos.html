Apresentação do Projeto

O projeto da nossa equipe é a contrução de um e-commerce para uma loja de roupas de atacado e varejo, o e-commerce foi desenvolvido para a empresa fazer a organização e vendas de seus produtos, nossa turma de desenvolvedores tem como instituião de ensio a escola SENAC-CASTANHAL-PA. Temos como docentes, o professor Gabriel Costa, que ministra as aulas do dia-a-adia, e o professor Ertonilson Rocha que ministra as auas voltadas a criação do projeto integrador Pi. Nossa turma conta com de 12 Integrantes que trabalhoaram arduamente para criar este Projeto, veja ao lado a foto dos desevolvedores, e abaixo agumas das lingagem que foram ultilizadas para a construção do nosso E-commerce "Lojão do 20".

Tecnologias Utilizadas

  • HTML

    Em uma atividade de Desenvolvimento Web, o HTML ( HyperText Markup language) é uma linguagem de marcação utilizada para a criação de conteúdo de texto e outros elementos de uma página Web, como imagens, áudios, vídeos e links. O HTML é o corpo de uma págiana web, dentro dela podemos criar uma estrutura com títulos,parágrafos e etc...

  • CSS

    Apredemos sobre o CSS e sua funcionalidade e que ele é uma limguagem de estilo utilizada para a formação de uma apresentação de um documento HTML. Ele pode ser utilizada para formatar textos, tabelas, e outros elementos HTML. O CSS é o rosto do HTML, você pode olhar dessa forma para ser mas fácil entender sobre essa linguagem.

  • Python

    Python é uma linguagem de programação poderosa e versátil que é amplamente usada em muitas áreas, desde desenvolvimento web até análise de dados e inteligência artificial. Sua sintaxe simples, bibliotecas extensas e comunidade ativa tornam-na uma ótima escolha para desenvolvedores de todos os níveis.

  • Django

    Django é um framework web poderoso e flexível que é amplamente usado para desenvolver aplicações web complexas. Sua arquitetura MVC, desenvolvimento rápido, segurança e escalabilidade o tornam uma escolha popular entre os desenvolvedores. Embora possa ser complexo para aplicações muito simples, Django é uma ótima escolha para desenvolver aplicações web robustas e escaláveis.

  • MySQL

    MySQL é um sistema de gerenciamento de banco de dados relacional poderoso e flexível que é amplamente usado em várias aplicações. Sua facilidade de uso, comunidade ativa, custo baixo e integração com outras tecnologias o tornam uma escolha popular entre os desenvolvedores. Embora possa ter limitações de desempenho e segurança, MySQL é uma ótima escolha para muitas aplicações de banco de dados.
